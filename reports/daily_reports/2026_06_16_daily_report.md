### Какие 3-5 выводов сделал после чтения:
1. Если мы не знаем, сколько аргументов передаем в функцию, мы можем создать функцию, которая будет принимать произвольное количество аргументов, добавив * перед именем параметра.
2. При вызове функции мы передаем значения по умолчанию для параметров. Если при вызове функции не передавать аргументы, будут использоваться значения по умолчанию.
3. Если мы не возвращаем значение из функции, то по умолчанию наша функция возвращает None.
4. Чтобы вернуть значение из функции, мы используем ключевое слово return, за которым следует возвращаемая переменная.
