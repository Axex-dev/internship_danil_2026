### Инструкция к программе "анализатор учебных текстовых заявок" ###
### Назначение программы
    Программа предназначена для:
        1. записи пользовательских заявок в текстовый файл;
        2. анализа текста заявок на наличие ключевых слов;
        3. подсчёта статистики по найденным совпадениям;
        4. сохранения результатов анализа в отдельный файл.
### Используемые файлы
    Программа работает с тремя файлами:
        1. ../data/week_02_request_analysis_keywords.json — список ключевых слов (в формате JSON)
        2. ../data/week_02_request_analysis.txt — файл с введёнными заявками
        3. ../results/week_02_request_analysis_result.txt — файл с результатами анализа
### Как работает программа
    1. Ввод заявок
        При запуске программа запрашивает у пользователя количество заявок и сами заявки
        Все введённые строки сохраняются в файл: week_02_request_analysis.txt
    2. Обработка текста
        Для каждой строки выполняются следующие действия:
        текст приводится к нижнему регистру;
        выполняется поиск ключевых слов;
        определяется количество найденных совпадений.
    3. Поиск ключевых слов
        Программа сравнивает каждую заявку со списком ключевых слов из JSON-файла.
        Если ключевое слово встречается в тексте, оно добавляется в список найденных.
    4. Формирование результат
        Для каждой строки формируется отчёт вида.
    5. Итоговая статистика
        В конце выводится:
        общее количество обработанных строк;
        количество строк, содержащих ключевые слова.
    6. Сохранение результата
        Результаты сохраняются в файл:
        ../results/week_02_request_analysis_result.txt
### Ручной запуск программы

 ```bash
 python3 ../python_basics/week_02_request_text_analyzer.py    
 ```
